## Mobile Responsive Behavior

### Mobile (< 768px)
- Modal: Full width with reduced padding
- Repository list: Single column
- Buttons: Stack vertically
- Font sizes: Slightly reduced
- Button text: Abbreviated on very small screens

### Tablet (768px - 1024px)
- Modal: 90% width
- Repository list: 1-2 columns
- Buttons: Side by side
- Full text visible

### Desktop (> 1024px)
- Modal: 1000px max width
- Repository list: Full layout
- Buttons: Full width or side by side
- All content visible
